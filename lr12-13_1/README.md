Опис алгоритму

Програма відкриває файл number.txt і зчитує з нього числа. Для кожного числа перевіряє, чи воно від’ємне. Якщо так — обчислює його квадрат і додає до загальної суми. Після зчитування всіх чисел програма закриває файл і виводить суму квадратів від’ємних чисел.

Приклад виконання 

Зчитуємо числа з файлу number.txt...
Сума квадратів від'ємних чисел: 146
