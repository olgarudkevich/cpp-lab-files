Опис алгоритму 

Програма зчитує з файлу matrix.txt елементи квадратної матриці 5×5, виводить її на екран, обчислює суму додатних елементів, що знаходяться над головною діагоналлю, виводить отриману суму та записує результат у файл result.txt.
